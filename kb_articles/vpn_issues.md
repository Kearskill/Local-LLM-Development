# KB-102: VPN Connection Troubleshooting

## Overview
This guide covers common issues with the TechCorp GlobalConnect VPN.

## Common Error Codes
* **Error 404:** Peer not found. Usually caused by an outdated client version.
* **Error 802:** Authentication failed. This happens if your AD password expired or MFA was denied.

## Troubleshooting Steps
1. **Check Connectivity:** Ensure you are on a stable internet connection.
2. **Update Client:** Open 'Company Portal' and check for GlobalConnect updates.
3. **Reset Credentials:** If Error 802 persists, visit `https://reset.techcorp.com`.

## Escalation Policy
If the issue is not resolved after step 3, create a **High** severity ticket for the Network Team.
